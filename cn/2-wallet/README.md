## 更多资源
### 创建钱包
可以有多种方式创建钱包。

完全同步了区块链数据的，可以考虑
* NEO-GUI -> https://docs.neo.org/en-us/node/gui/install.html
* NEO-CLI -> https://docs.neo.org/en-us/node/cli/cli.html

对于不需要进行同步的轻客户端，可以考虑
* O3 钱包-> https://o3.network/
* NEON 钱包-> https://neonwallet.com/

你可以在相关的钱包链接中查看更详细的使用指南
